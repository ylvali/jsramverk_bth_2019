Installera moduler

NgModules används för att installera moduler : @NgModule.

Modulen används för att kompilera komponenters template, för att exportera användning till det publika samt för att injicera dependencys.

Angulars dokumentation har en bra beskrivning: Angular dokumentation: NgModules

Starta appen

Angular CLI, Node.js & npm används. För mera info: Angular dokumentation: Setup
