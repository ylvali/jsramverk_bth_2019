Subprotocols (Broadcast/Echo)

Select what subprotocol to use when connecting to an server. Each server supports its own variants of subprotocols.

  • ws://dbwebb.se:1337/ supports broadcast-protocol, echo-protocol/none.
  • ws://demos.kaazing.com/echo support no subprotocols.
  • ws://localhost:1337/ supports text or json

Open up devtools and see output in the console and network tab.
